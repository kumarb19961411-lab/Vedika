# User Onboarding Guide

## Flow Description
Users verifying OTP for the first time must provide minimal details.

## Steps
1. **Personal Details:** Name, Wedding/Event Date (optional).
2. **Commit:** Data is written to `users/{uid}`.

## Navigation Exit
User is routed to `UserShell`.
